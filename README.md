# 🇮🇱 Israel Alerts Data Analysis & Visualization

**פורטל נתוני חירום ואזעקות - ישראל**

פרויקט זה מרכז ניתוח נתונים מקיף, ויזואליזציות ומפות אינטראקטיביות של אירועי ביטחון ואזעקות בישראל, בדגש על מלחמת "חרבות ברזל".

## 📊 מה בפרויקט?

הפרויקט כולל דאשבורד מרכזי המאגד מספר סוגי ניתוחים:

1.  **מפת איומים אינטראקטיבית (Live Dash App):**
    * אפליקציה חיה המאפשרת פילוח אזעקות לפי תאריכים, שעות, סוגי איום ומדינות מקור (עזה, לבנון, איראן ועוד).
    * מבוססת על `Dash` ו-`Plotly`.

2.  **דוחות סטטיים וניתוחי עומק:**
    * **מפת חום (Heatmap):** פיזור גיאוגרפי של האזעקות.
    * **מירוץ הערים (Bar Chart Race):** אנימציה המציגה את הערים המטווחות ביותר לאורך זמן.
    * **ניתוח יומי וחודשי:** מגמות ירי לאורך ציר זמן.
    * **התפלגות שעות:** באילו שעות ביממה מתרחשים רוב האירועים.

## 🛠️ טכנולוגיות וכלים

הפרויקט נכתב ב-Python ועושה שימוש בספריות הבאות:
* **Pandas:** עיבוד וניקוי נתונים (`alarms.csv`, `coord.csv`).
* **Plotly & Dash:** יצירת גרפים אינטראקטיביים ובניית ממשק ה-Web.
* **Jupyter Notebook:** מחקר וניתוח ראשוני (`project.ipynb`).

## 📂 מבנה הקבצים

* `app.py` - קוד המקור לאפליקציית הדאשבורד (Dash).
* `index.html` - עמוד הנחיתה המרכזי של האתר.
* `project.ipynb` - מחברת המחקר עם תהליכי עיבוד הנתונים.
* `*.html` - דוחות ויזואליזציה שנוצרו (מפות, גרפים).
* `alarms.csv` - קובץ הנתונים הגולמי.

## 💻 הרצה מקומית (Installation)

אם ברצונך להריץ את קוד ה-Python אצלך במחשב:

1.  שכפל את המאגר:
    ```bash
    git clone [https://github.com/harelgam/Israel_alarms.git](https://github.com/harelgam/Israel_alarms.git)
    ```
2.  התקן את החבילות הנדרשות:
    ```bash
    pip install -r requirements.txt
    ```
3.  הרץ את האפליקציה:
    ```bash
    python app.py
    ```
תודות ליובל הרפז על הדאה המאורגן מפורטל החירום הלאומי דרך צבע אדום
